# C - 差值为1 (CSES - 1070)

如果一个整数排列 $1, 2, \dots, n$ 中不存在相邻元素之差为 1 的情况，则称该排列为美丽排列。

给定 $n$，请构造一个美丽排列（如果存在这样的排列）。

## 输入

唯一输入行包含一个整数 $n$。

## 输出

输出一个由整数 $1, 2, \dots, n$ 组成的美丽排列。若存在多个解，可输出其中任意一个。若无解，则输出 "NO SOLUTION"。

## 约束条件

* $1 \le n \le 10^6$

## 示例

**示例 1**
```text
Input:
5

Output:
4 2 5 3 1
```

**示例 2**
```text
Input:
3

Output:
NO SOLUTION
```
