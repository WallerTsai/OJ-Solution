# E - 异或最大值 (CodeChef - DRAGNXOR)

给定两个 $N$ 位非负整数 $A$ 和 $B$（二进制表示恰好 $N$ 位，不足补前导 $0$）。定义 $A$ 的一个“位乱序” $A'$ 为：通过重新排列 $A$ 的二进制位得到的任意 $N$ 位整数（保持位数 $N$ 不变，但可以任意调换各个位的位置）。类似定义 $B'$。

求在所有可能的 $A'$ 和 $B'$ 中，$A'$ 与 $B'$ 的按位异或（xor）的最大可能值。

## 提示

* 异或运算：对应位相同时得 0，不同得 1。例：$3(011)_2 \text{ xor } 5(101)_2 = 6(110)_2$
* 注意 $A$ 和 $B$ 的二进制表示可能有前导 $0$，总位数为 $N$。

## 输入格式

第一行一个整数 $T$，表示测试用例的数量。
接下来 $T$ 行，每行三个整数 $N, A, B$，其中 $N$ 表示位数，$A$ 和 $B$ 是给定的整数 ($0 \le A, B < 2^N$)。

## 输出格式

对于每个测试用例，输出一行一个整数，表示最大可能的 $A' \oplus B'$ 的值。

## 数据范围

* $1 \le T \le 100$
* $1 \le N \le 30$
* $0 \le A, B < 2^N$

## 示例

```text
Input:
3
3 5 4
5 0 1
4 3 7

Output:
7
16
14
```

* **案例 1**：5 和 4 的 3 位二进制分别是 $(101)_2$ 和 $(100)_2$。重新排列后，异或最大值是 $(110)_2 \oplus (001)_2 = (111)_2 = 7$
* **案例 2**：最大结果是 $(00000)_2 \oplus (10000)_2 = (10000)_2 = 16$
* **案例 3**：最大结果是 $(0011)_2 \oplus (1101)_2 = (1110)_2 = 14$
