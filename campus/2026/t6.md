# F - 分糖果 (CodeChef - WCEPROB7)

有 $N$ 个学生和 $K$ 颗糖果。第 $i$ 个学生有一个上限 $a_i$，表示该学生最多可以接收 $a_i$ 颗糖果。糖果必须全部发完，不能有剩余。

求分配糖果的方式数。两种分配方式不同当且仅当至少有一个学生收到的糖果数不同。结果对 $10^9 + 7$ 取模。

## 输入格式

第一行：两个整数 $N$ 和 $K$。
第二行：$N$ 个整数 $a_1, a_2, \dots, a_N$。

## 输出格式

输出分糖果的方案数，对 $10^9 + 7$ 取模。

## 约束条件

* $1 \le N \le 100$
* $0 \le K \le 10^5$
* $0 \le a_i \le K$

## 示例

**Input**
```text
3 4
1 2 3
```

**Output**
```text
5
```

**说明**：分糖果的方案有五种，具体如下：
(0,1,3)、(0,2,2)、(1,0,3)、(1,1,2)、(1,2,1)。
这里，每个序列的第 i 个数字表示第 i 个孩子拿到的糖果数。
